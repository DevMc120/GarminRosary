# GarminRosary

**GarminRosary** is a standalone rosary companion application for Garmin devices. It helps users pray the Rosary without needing a phone, offering a simple and focused interface on the wrist.

## About the Project:
This project is a transparent experiment in AI-assisted development.

We believe in open source and the spiritual value of sharing tools freely. Whether you are a developer looking to learn Monkey C or a user wanting to improve the app, you are welcome here.

## Features
* **Offline:** Works entirely without phone connection.
* **Privacy First:** No data tracking, no GPS usage.
* **Simple Interface:** Designed for distraction-free prayer.
